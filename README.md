# PantryKal

Calorie & pantry tracker for busy professionals. Log meals, track macros against a daily target, and get meal suggestions based on what's actually in your kitchen. Pantry stock auto-depletes as you log meals and builds your shopping list automatically.

## Stack
- React 18 (Vite)
- lucide-react icons
- Tailwind CSS (CDN)
- Hosted on Vercel

## Run locally
npm install
npm run dev
